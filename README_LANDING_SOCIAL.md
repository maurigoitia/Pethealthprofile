# Pessy Landing Pages for Social Media Campaigns

This branch contains specialized landing pages for social media marketing campaigns and promotional content.

## Contents

- `src/app/pages/LandingSocialPage.tsx` - Main social media landing page
- `src/app/pages/EmpezarLandingPage.tsx` - "Get Started" landing page
- `src/assets/` - Images and assets used by landing pages
- `public/team/` - Team photos and promotional images

## Purpose

These pages are designed to:
- Capture leads from social media campaigns
- Provide compelling CTAs (Call-To-Action)
- Showcase team and product features
- Direct users to sign up or learn more

## Usage

These pages can be served as standalone routes or embedded within the main Pessy PWA application. They are optimized for:
- Mobile responsiveness
- Social sharing
- SEO and meta tags
- Conversion tracking

## Components

### LandingSocialPage.tsx
Main landing page for social media acquisition campaigns. Features promotional content, call-to-action buttons, and social proof elements.

### EmpezarLandingPage.tsx
"Get Started" landing page that guides new users through the initial onboarding process and sign-up flow.
