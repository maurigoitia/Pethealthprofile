## ⚠️ ÁREAS DE MEJORA

### CRÍTICAS (Impacto Alto)

#### 1. **Open Graph Image - Genérica**
**Problema:**  
Se usa /og-cover.png para todos los posts, impactando el CTR en redes sociales

**Solución:**  
```html
<!-- ACTUAL -->
<meta property="og:image" content="https://pessy.app/og-cover.png">

<!-- RECOMENDADO -->
<meta property="og:image" content="https://pessy.app/og/blog/frecuencia-bano.jpg">
<meta property="og:image:width" content="1200">
<meta property="og:image:height" content="630">
<meta property="og:image:type" content="image/jpeg">
```

**Impacto:** Aumentar CTR en redes sociales hasta un 20-30%

---

#### 2. **Breadcrumb Schema Markup - Ausente**
**Problema:**  
Breadcrumb HTML presente pero sin JSON-LD schema

**Solución:**  
```json
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "name": "Blog",
      "item": "https://pessy.app/blog"
    },
    {
      "@type": "ListItem",
      "position": 2,
      "name": "Tips",
      "item": "https://pessy.app/blog?category=tips"
    },
    {
      "@type": "ListItem",
      "position": 3,
      "name": "¿Cada cuánto bañar a mi perro?"
    }
  ]
}
```

**Impacto:** Mejora visibilidad en SERPs (breadcrumb rich snippet)

---

#### 3. **FAQ Schema - No Implementado**
**Problema:**  
Contenido tipo Q&A sin FAQPage schema

**Solución:**  
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "¿Cada cuánto bañar a mi perro?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Para la mayoría de los perros, entre una vez cada 3 y una vez cada 8 semanas..."
      }
    },
    {
      "@type": "Question",
      "name": "¿Qué factores influyen en la frecuencia de baño?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "El tipo de pelaje, nivel de actividad, salud de la piel y estación del año..."
      }
    }
  ]
}
```

**Impacto:** Posible aparición en FAQ rich snippets
