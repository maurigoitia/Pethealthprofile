Sobre las Animaciones

Cada personaje incluye animaciones CSS puras: float (flotación suave), breathe (respiración), wiggle (orejas/aletas), blink (parpadeo) y tailwag (cola). Los SVG son archivos standalone, importables directamente en React con Vite o como <img> tags. Zero dependencias — solo CSS nativo.
