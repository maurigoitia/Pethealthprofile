# Pessy Website

This branch contains the standalone landing page for pessy.app.

## Contents

- `public/landing.html` - Main landing page for the website
- `public/pessy-logo.svg` - Pessy logo in SVG format
- `public/robots.txt` - SEO robots configuration
- `public/sitemap.xml` - XML sitemap for search engines
- `firebase.json` - Firebase hosting configuration
- `.firebaserc` - Firebase project configuration
- `deploy-with-landing.sh` - Deployment script

## Deployment

To deploy the website to Firebase Hosting:

```bash
./deploy-with-landing.sh
```

This will deploy the landing page to pessy.app.
